IP Routing Nedir , Çeşitleri Nelerdir ?

Haziran 22, 2018 / by Erdem Yıldız

Merhaba arkadaşlar bugün sizlere Ip Routing ve çeşitler hakkında açıklama yapıp daha sonra türlerinden bahsedeceğim. Haydi bir giriş yapalım...

Router'ların temel görevi data'ları yönlendirme yapmasıdır.Bunu yaparken'de Routing Table (Yönlendirme tablosu) baz alınarak haraket eder.Routing Table üzerinde Static Routing ve Dinamic Routing komutları yazılır. Data'lar bir network'den başka bir network'e Routing Table üzerinde Static Routing yada Dinamic Routing kurallarına bakarak aktarılır

Static Routing

Birkaç adet Router'ın bulunduğu network topolojilerinde Router'lar arası data akratımının manuel olarak yapıldığı yöntemdir.Eğer tüm topolojide Static Routing yapılacaksa her bir router için bu tanımlama yapılmalıdır.

  • Static Routing Yapılandırması Daha fazlası için tıklayınız<

Dinamic Routing

Router'ların kendi aralarında network routing tablolarını diğer router'lara paylaştığı bir dil olarak düşünülebilir.Demek istediğim router'lar kendi aralarında hangi cihazın kime ait olduğu ve nasıl gidileceği bilgisini birbiri arasında paylaşır ve bunun yanında router'ların bağlı olduğu yol'da bir problem çıktığında bunu diğer router'lar'la paylaşarak problem'li yolun aralarında alternatifi bulunur.Static routing'den farkını göstermek gerekirse A router ile B router'u arasında bir kural yazıldığını farz edin.Eğer bu 2 router arasındaki yolda bir problem çıkarsa data aktarımı gerçekleşmez çünkü static routing alternatif bir yol aramaz.
